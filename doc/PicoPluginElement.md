## Usage

In the .md file, wrap the content with a start tag and an end tag. Just like:

```
<!--pico-element xxx-->
    ... ...
<!--pico-element-end-->
```
- xxx is the name of element. 
- This theme requires jQuery, but it will load automatically.

## Elements

### card-1

__# Code__
```
<!--pico-element card-1-->
** card-1 **  

A 100% wide card.  

[ More ... ](#)  
<!--pico-element-end-->
```
